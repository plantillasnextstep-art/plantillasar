# PlantillaAR

Generador de plantillas ortopédicas 3D personalizadas, con historial de pacientes y comparativa de evolución.

## Funciona sin internet
Todos los datos de pacientes se guardan en la computadora donde se usa, en el navegador. No depende de conexión para trabajar.

## Cómo actualizar
Cuando hay una nueva versión publicada, la app muestra un aviso arriba con un botón "Actualizar ahora".

## Estructura
- `index.html` — la aplicación completa
- `manifest.json` — configuración para instalar como app (PWA)
- `sw.js` — service worker, permite funcionamiento offline y actualizaciones
- `icon-192.png` / `icon-512.png` — íconos de la app
